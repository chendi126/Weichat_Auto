# 星际电报风格设计说明

## 🌌 设计理念

**"让每一篇文章都像一份来自未来的星际电报，理性而不失浪漫"**

这是一种独特的内容呈现方式，将科技新闻与太空美学相结合，创造出既专业又富有诗意的阅读体验。

---

## ✨ 视觉特征

### 1. 极简留白
- 大量留白空间，营造呼吸感
- 行距设置为2.0，提供舒适的阅读体验
- 内边距充足，让内容有"漂浮"的感觉

### 2. 几何线条
- **顶部线条**：从透明到金色到橙色渐变
- **底部线条**：从透明到红色到金色渐变
- 线条高度3px，精细而不突兀
- 作为视觉引导，如同星际航线

### 3. 星际点缀
- 20颗星星均匀分布在页面各处
- 每颗星星都有光晕效果（box-shadow）
- 透明度0.7-0.9，营造闪烁感
- 大小2-3px，精致而不喧宾夺主

### 4. 温暖光色
- **背景色**：深蓝渐变 (#0a0a1a → #1a1a2e → #16213e)
- **主色调**：金色 (#ffd700)
- **辅助色**：橙色 (#ff8c00)、红色 (#ff6b6b)
- **文字色**：温暖米色 (#f0e6d3)

## 📐 页面结构

### 顶部区域
```
┌─────────────────────────────────────┐
│  ◆ INTERSTELLAR TRANSMISSION ◆    │  ← 小标题，大写，金色
│  [标题区域 - 渐变色大标题]           │
│  ─── 2026.03.09 ───                │  ← 装饰线 + 日期
└─────────────────────────────────────┘
```

### 内容区域
```
┌─────────────────────────────────────┐
│  文章内容：                          │
│  - 字号：18px                        │
│  - 行距：2.0                         │
│  - 对齐：两端对齐                    │
│  - 段落间距：25px                    │
│  - 小标题：橙色，28px                │
│  - 重点文字：金色加粗                │
└─────────────────────────────────────┘
```

### 底部区域
```
┌─────────────────────────────────────┐
│  ─────────────────────────────────  │  ← 分隔线
│  ◆ END OF TRANSMISSION   ●●●        │
└─────────────────────────────────────┘
```

## 💡 使用建议

### 适用场景
✅ 微信公众号文章  
✅ 科技类博客  
✅ 电子出版物  
✅ 产品说明文档  
✅ 新闻资讯网站  

### 不适用场景
❌ 需要大量表格的文档  
❌ 严肃的学术论文  
❌ 需要打印的文档  

### 内容建议
- 标题控制在20字以内
- 正文分段清晰，每段不超过200字
- 适当使用小标题分割内容
- 重点内容用金色加粗标注

---

## 🔧 自定义修改

如果需要调整样式，可以修改 `src/graphs/nodes/write_article_node.py` 中的以下函数：

- `generate_interstellar_html()` - 整体布局
- `generate_stars()` - 星星数量和位置
- `markdown_to_html()` - 内容转换规则
